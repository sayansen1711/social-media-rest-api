# Social Media REST API

A demo social media backend built using Spring Boot and Gradle.

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- JWT Authentication
- Flyway

## Features
- User signup & login
- Create posts
- Like & comment on posts
- Follow users
- Secure REST APIs

## Architecture
- Controller → DTO → Service → Repository → Entity
- JWT based authentication
- Role-based access

```bash
./gradlew bootRun
